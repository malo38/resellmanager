✅ Méthode recommandée — synchronisation automatique toutes les 5 min
1
Téléchargez le ZIP de l'extension Vinted Manager depuis le lien envoyé par votre administrateur, dézippez-le sur votre bureau.
2
Dans Chrome, allez sur chrome://extensions, activez le Mode développeur (en haut à droite), puis cliquez "Charger l'extension non empaquetée" et sélectionnez le dossier dézippé.
3
Cliquez sur l'icône 🔗 Vinted Manager dans Chrome, connectez-vous avec votre email/mot de passe Vinted Manager (pas Vinted), puis ouvrez un onglet vinted.fr.
4
Cliquez "Synchroniser maintenant" dans l'extension. Vos articles Vinted apparaîtront automatiquement sur ce site !
⚠️ Méthode manuelle — le cookie expire tous les 7-14 jours
1
Ouvrez vinted.fr dans Chrome et connectez-vous à votre compte Vinted.
2
Appuyez sur F12 (ou Cmd+Option+I sur Mac) pour ouvrir les outils développeur.
3
Cliquez sur l'onglet "Application" → dans le menu gauche "Cookies" → "https://www.vinted.fr".
4
Trouvez le cookie nommé _vinted_fr_session, cliquez dessus et copiez toute la valeur (très longue).
5
Collez la valeur ci-dessous et cliquez "Enregistrer".
